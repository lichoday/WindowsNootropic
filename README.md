# WindowsNootropic
A raw PowerShell script to debloat Windows 10/11, kill tracking, and free up system resources.

What it does:
Disables telemetry, Cortana, and spy services
Nukes pre-installed bloatware (Xbox, Bing, etc.)
Stops background apps from draining memory
Disables Game DVR and hibernation
Clears system cache and temp files

How to use:
Download nootropic.ps1
Open PowerShell as Admin
Run: Set-ExecutionPolicy Bypass -Scope Process -Force; .\WindowsNootropicByLichoday.ps1
Reboot your PC
